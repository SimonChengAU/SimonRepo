1>How to test the Get Current Weather demo software. 
	Maven 3.3.3 is recommended. 
	Change the current directory to the WeatherWebApp directory download from the Github. 
	Run:
		mvn test
2>How to run the Get Current Wwather demo software.
	Maven 3.3.3 is recommended. 
	Change the current directory to the WeatherWebApp directory download from the Github. 
	Run:
		mvn package
	Then run:
		mvn tomcat:run
	Then open a browser and type in URL: http://localhost:8080/WeatherWebApp/
